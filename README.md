INSTALAÇÃO


1- Baixe e instale o XAMPP. Atente para o local de instalação!https://www.apachefriends.org/pt_br/index.html

2- Baixe ou clone o repositório do git. https://github.com/LuizDoPc/psCompJuniorFase2

3- Mova a pasta do repositório para a pasta do XAMPP (o local padrão é C:\xampp\htdocs)

4- Abra o XAMPP e dê start no apache e no mysql.

5- Abra o arquivo "banco de dados.sql" com o bloco de notas.

6- Copie todo o conteúdo do arquivo.

7- Abra seu navegador e vá até http://localhost/phpmyadmin/

8- No menu superior clique em SQL.

9- Cole o conteúdo do arquivo na caixa de texto.

10- Clique em executar.

11- No seu navegador, vá até C:\xampp\htdocs\psCompJuniorFase2\visao

12- Agora você pode editar o código fonte do site!

13- Após terminar de editar, envie suas mudanças para o GitHub.



COMO ENVIAR SUAS MUDANÇAS PARA O GITHUB

1- Baixe e instale o git https://git-scm.com/downloads

2- Na pasta do projeto, clique com o botão direito.

3- Clique em GIT Bash Here

4- No terminal, use git config --global user.name "seunome"

5- No terminal, use git config --global user.email seuemaildogit@example.com

6- No terminal, use git add --all

7- No terminal, use git commit -m "explique aqui o que você fez no código"

8- No terminal, use git push origin master

9- Pronto!
